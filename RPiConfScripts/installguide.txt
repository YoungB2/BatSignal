Documents the installation process for batman and RPi node network configurations:

piome-network is an init.d startup script which will need to be copied to the /etc/init.d/ directory and given the execute privileges. 
net_interfaces is a configuration file which needs to be moved to the /etc/network/ directory and renamed "interfaces".

piome-network script setup:
    >cp piome-network /etc/init.d/bat-signal
    >sudo chmod +x /etc/init.d/bat-signal
    >sudo update-rc.d /etc/init.d/bat-signal defaults		#This configures the script to run on boot

interfaces configuration:
    >cp net_interfaces /etc/network/interfaces
